# Dashboard Fragantica_Perfumes (2020–2024)


![Uploading Animation2.0-ezgif.com-optimize.gif…]()

Para este proyecto se seleccionó el sector de perfumería y fragancias. El objetivo del análisis es comprender el comportamiento del mercado a partir de un conjunto de datos que contiene información sobre perfumes, marcas, calificaciones, interés de compra y características olfativas.

## 📁 Estructura

| Archivo | Descripción |
|---------|-------------|
| Tablero/Proyect_Perfumes | Dashboard interactivo |
| Docs/Documentacion_Tecnica.pdf | Detalle técnico del modelo |
| Docs/MEDIDAS_DAX.md | Medidas DAX utilizadas |
| images/ | Capturas del modelo y reportes 

## 📐 Medidas DAX

Incluye:
- Total Votos, Clasificacion Promedio y Mediana de Votos
- Media de Votos y Filtros/Segmentadores
- Forescast y Linea de Tendencia
